# Proyecto de Infraestructura Virtual

Este repositorio ha sido creado con el fin de alojar el proyecto de la asignatura Infraestructura Virtual.

## Descripción del proyecto

En este proyecto se desarrollará una plataforma que haga dietas para los usuarios de la plataforma, esta plataforma busca solventar los problemas que tienen algunas personas para llevar una dieta equilibrada, tambien puede servir para deportistas que busquen una dieta específica (como por ejemplo rica en proteínas). El modelo de negocio será por medio de la suscripción a esta plataforma.
